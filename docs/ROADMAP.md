# ShieldOS Roadmap

## Version 0.2 - Daily Driver VM

Current target:

- Bootable NixOS 26.05 ISO
- KDE Plasma desktop
- Flatpak/Flathub app support
- Steam/Proton gaming base
- Wine/Winetricks for Windows app experiments
- Developer tools
- Privacy defaults
- VM install helper

## Version 0.3 - Better installer

Planned:

- Polished graphical installer
- Better first-boot setup
- Privacy level selection
- User-created account instead of fixed demo account
- Encrypted install option

## Version 0.4 - Shield apps

Planned:

- Shield Updater UI
- Shield Privacy Dashboard improvements
- Shield Game Center
- Shield App Hub that explains Flatpak/Nix/Wine options

## Version 1.0 - Real daily driver

Required before 1.0:

- Clean install flow
- Strong update/rollback docs
- Hardware testing
- NVIDIA/AMD/Intel GPU guide
- Laptop power testing
- Printer/Bluetooth testing
- Security review
