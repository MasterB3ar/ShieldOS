# ShieldOS Roadmap

## Version 0.1 - Developer Preview

Goal: make a bootable, testable live ISO.

Included:

- NixOS 26.05 base
- KDE Plasma 6
- Shield Center prototype
- Firewall/AppArmor/auditd
- Gaming and developer packages
- Private Mode helper

## Version 0.2 - Installable Preview

Goal: make ShieldOS usable on a spare drive.

Add:

- Calamares or custom installer profile
- Full-disk encryption default
- Btrfs snapshots
- Secure rollback boot entries
- Proper first-boot user creation
- No hardcoded live password on installed systems

## Version 0.3 - Real Privacy Dashboard

Goal: make privacy visible and easy.

Add:

- Camera/microphone access indicators
- Network activity per app
- Flatpak permission viewer/editor
- Recent file access summary
- Background app list
- Privacy score

## Version 0.4 - ShieldOS Design System

Goal: make the OS look unique.

Add:

- Plasma theme
- Icon theme override
- Wallpapers
- Boot splash
- Login screen theme
- Sound scheme
- Better welcome app

## Version 0.5 - App Store

Goal: make app installation easy and safe.

Add:

- Flatpak-first app store UI
- Privacy labels per app
- Recommended apps
- Gaming category
- Developer category

## Version 1.0 - First Public Release

Goal: safe enough for real daily testing.

Requirements:

- Reproducible ISO build
- Encrypted installer
- Automatic rollback after failed update
- Signed release checksums
- Basic docs site
- Hardware testing list
